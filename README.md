# RoleModel Rmr

## How do I install these formulae?

`brew install rolemodel/rmr/<formula>`

Or `brew tap rolemodel/rmr` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "rolemodel/rmr"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
